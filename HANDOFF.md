### 각 페이지 공통 섹션 구조

1. **Navbar** — 로고(`/`) + 상담신청 버튼(`/apply`), 스크롤 시 흰 배경 전환
2. **히어로** — 네이비 그라디언트 배경, 메인/서브 카피, 통계 3개, CTA 버튼
3. **학점 취득 구조** — 온라인 수업·자격증·독학사 카드 3개 + 기존 학점 배너
4. **과정별 플랜 / 콘텐츠 섹션** — 페이지별 상이
5. **자격증 안내** — 카드형 배치
6. **FAQ** — 아코디언 (ChevronDown 토글)
7. **하단 CTA** — 네이비 배경, 무료 상담 신청 → `/apply`
8. **푸터** — 사명 + 저작권
